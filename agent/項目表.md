# 項目表

| 路徑 | 用途 |
| --- | --- |
| `src/` | SSH server 核心程式碼 |
| `src/shell-runner.js` | 互動 shell 與真 PTY lifecycle/resize 處理 |
| `scripts/` | 初始化、密碼雜湊、host key 產生工具 |
| `config/` | 使用者與命令白名單設定範本 |
| `s12ryt/` | SSH shell / SFTP 使用者工作目錄（不提交） |
| `keys/` | SSH host key（不提交） |
| `agent/` | 專案任務與維護記錄 |
